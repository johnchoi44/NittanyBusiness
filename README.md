# NittanyBusiness
CMPSC 431 Database Management Project

## Requirements

* node
* sqlite3


## Project Structure
The code is organized into three main modules:

1. **frontend/**: Frontend
2. **backend/**: Backend
3. **.idea/**: Data Source

## Usage

### Quick Start
```
cd frontend
npm start

cd ..
cd backend
node server.js

```
